User interface

  • Double click on canvas to open the selector.
  • Type into the selector then press ENTER to create a new node.
  • Drag from a node output to a node input to create a link.
  • Drag on canvas to select items.
  • Click on item to select it.
  • Click while pressing CTRL or CMD to enable multi selection.
  • Drag selected nodes to translate them.
  • Press BACKSPACE to delete selected items.
  • Move pointer while holding SPACE to translate viewport.
